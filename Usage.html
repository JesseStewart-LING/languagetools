¿Cómo se usa este diccionario?
Este diccionario usa la ortografía original del español con modificaciones que se encuentran en Imbabura, por lo que se recomienda evitar la búsqueda utilizando el quichua unificado. Existen muchas variaciones en la ortografía, entonces sino encuentra la palabra deseada, intente con las variaciones mencionadas en las tablas de arriba.

How to use this dictionary?
This dictionary uses the original Quichua spelling based on Spanish (modified for Imbabura Media Lengua & Quichua); avoid using unified Quichua. There is a lot of variation in the spelling used herein, therefore, if you do not find the word you are looking for on the first try, consider using one or more of the variations found on the above tables.
